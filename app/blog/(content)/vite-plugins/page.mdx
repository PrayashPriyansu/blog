# Vite Plugins: Power Up Your Workflow

[Vite](https://vitejs.dev/) is a super-fast build tool for modern web apps. Its **plugin system** lets you add or customize features‚Äîeverything from React support to [PWA](https://web.dev/explore/progressive-web-apps)(Progressive Web Apps) capabilities. Here‚Äôs a **3-minute** rundown.

---

## Using Plugins

Install plugins with

```sh
npm install pluginName
```

or

```sh
yarn add pluginName
```

Then add them in <ILH>vite.config.js</ILH>:

```js
import { defineConfig } from "vite";
import react from "@vitejs/plugin-react";

export default defineConfig({
  plugins: [react(), pluginName()],
});
```

**That‚Äôs it!** Vite automatically applies the plugin in dev and build.

---

## Popular Plugins

1. **@vitejs/plugin-react**

- **What:** Official React Fast Refresh & optimized build.
- **Install:** `npm i @vitejs/plugin-react`

2. **vite-plugin-pwa**

- **What:** Quick PWA setup with offline caching.
- **Install:** `npm i vite-plugin-pwa`

3. **vite-plugin-svgr**

- **What:** Import SVGs as React components.
- **Install:** `npm i vite-plugin-svgr`

## How to use these plugins

```js
// vite.config.js
import { defineConfig } from "vite";
import react from "@vitejs/plugin-react";
import svgr from "vite-plugin-svgr";
import { VitePWA } from "vite-plugin-pwa";

export default defineConfig{
  plugins: [svgr(), VitePWA(), react()],
};
```

## Creating a Custom Plugin

A Vite plugin is just an object with **hooks**. For example:

```js
// my-plugin.js
export default function myPlugin() {
  return {
    name: "my-plugin",
    transform(code, id) {
      if (id.endsWith(".js")) {
        return `console.log("Hello!");\n` + code;
      }
      return code;
    },
  };
}
```

Add it to your config:

```js
import myPlugin from "./my-plugin.js";
export default {
  plugins: [myPlugin()],
};
```

Now every <ILH>.js</ILH> file will log ‚ÄúHello!‚Äù in dev mode.
